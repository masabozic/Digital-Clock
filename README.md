# Digital-Clock

Ovaj projekat predstavlja implementaciju digitalnog sata u programskom jeziku C, korišćenjem razvojnog okruženja Keil µVision.
Razvijen je u okviru predmeta Mikroprocesori, sa ciljem primene osnovnih koncepata ugrađenih sistema i rada sa vremenski zavisnom logikom.

Sat prikazuje vreme u standardnom formatu i simulira osnovne funkcionalnosti digitalnog sata.
